  <action>
Create `pf/optimizer.py` with the following:

**`split_slide(slide: dict) -> list[dict]`** тАФ Main entry point. Takes a slide dict and returns a list of 1+ slide dicts where each fits within USABLE_HEIGHT.

Algorithm:
1. Check if slide layout is in COLUMNAR_LAYOUTS (two-column, three-column, data-table, stat-grid). If not, return `[slide]` unchanged.
2. Use `LayoutAnalyzer.analyze_slide(slide, 0)` to check if the slide overflows. If no overflow, return `[slide]` unchanged.
3. For `two-column`: extract `left` and `right` block lists from `slide["data"]`. Call `_fit_split(left)` and `_fit_split(right)` to get (fits, remainder) tuples. Build slide_a with the fitting parts and slide_b with the remainders. Append " (cont.)" to slide_b's subtitle (or set subtitle to "(cont.)" if none exists).
4. For `three-column`: extract `columns` list. For each column, call `_fit_split(column)`. Build slide_a/slide_b similarly.
5. For `data-table`: extract `sections` list. Split at the section level (first N sections that fit, rest in slide_b).
6. For `stat-grid`: treat like two-column using `columns`.
7. Filter out slides that have no content using `_has_content(slide)`.
8. Return the list of non-empty slides.

**`_fit_split(blocks: list[dict]) -> tuple[list[dict], list[dict]]`** тАФ Given a list of content blocks, return (fits, remainder) where `fits` is the longest prefix that doesn't exceed USABLE_HEIGHT.

Implementation:
- Walk blocks, accumulating height via `LayoutAnalyzer.estimate_block_height(block)` + COLUMN_GAP between blocks.
- When adding the next block would exceed USABLE_HEIGHT, split at that index.
- If the FIRST block alone exceeds USABLE_HEIGHT, return `(blocks, [])` тАФ don't split (emit the oversized block as-is rather than producing an empty slide).

**`_has_content(slide: dict) -> bool`** тАФ Returns True if the slide has at least one non-empty column/section. Checks `left`, `right`, `columns`, `sections` in `data`.

Use `import copy` for `copy.deepcopy(slide)` when creating slide_a and slide_b to avoid mutating the original.

Import from `pf.analyzer`: `LayoutAnalyzer`, `USABLE_HEIGHT`, `COLUMN_GAP`, `COLUMNAR_LAYOUTS`.
