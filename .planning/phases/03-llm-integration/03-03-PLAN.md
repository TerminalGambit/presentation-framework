**Step 2: Create pf/sanitize.py** — XSS sanitization module:

```python
"""Sanitize LLM-generated content before it reaches Jinja2 templates."""
```

Define constants:
- `ALLOWED_TAGS = ["b", "i", "em", "strong", "a", "br", "p", "ul", "li", "code", "pre"]`
- `ALLOWED_ATTRS = {"a": ["href", "title"]}`

Define `safe_llm_text(value: str) -> str`:
- Import `bleach` with try/except ImportError fallback
- If bleach available: `return bleach.clean(value, tags=ALLOWED_TAGS, attributes=ALLOWED_ATTRS, strip=True)`
- If bleach NOT available: use a simple regex fallback that strips `<script>` and `<style>` tags: `re.sub(r'<(script|style|iframe|object|embed)[^>]*>.*?</\1>', '', value, flags=re.DOTALL | re.IGNORECASE)` then also strip remaining `<script` etc opening tags without closing. This ensures basic safety even without bleach installed.

Define `sanitize_slide_data(data: dict) -> dict`:
- Recursively walk a slide data dict
- For every string value, apply `safe_llm_text()`
- For lists: recurse into each element
- For dicts: recurse into each value
- Return the sanitized copy (do NOT mutate input)

This module must NOT import from `pf.builder` — it is a standalone utility.
