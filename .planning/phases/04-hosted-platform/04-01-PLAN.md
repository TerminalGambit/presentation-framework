<task type="auto">
  <name>Task 1: Add _rewrite_asset_paths() and base_url parameter to builder</name>
  <files>pf/builder.py, tests/test_builder.py</files>
  <action>
1. Add a module-level function `_rewrite_asset_paths(html: str, base_url: str) -> str` to `pf/builder.py`. This uses `re.sub` to match `(href|src)=["']<relative_path>["']` and prepend the base_url (stripped of trailing `/`). The regex MUST skip paths starting with `http://`, `https://`, `data:`, `//`, or `#` using a negative lookahead. Pattern: `r'(href|src)=(["\'])(?!http://|https://|data:|//|#)([^"\']+)\2'`. The replacement produces `{attr}={quote}{base}/{path}{quote}`.

2. Modify the `build()` method signature to accept `base_url: str | None = None`. After all HTML files are written (after `(out / "present.html").write_text(...)` and the theme copy block), add a post-process pass:
```python
if base_url:
    for html_file in out.glob("*.html"):
        html = html_file.read_text(encoding="utf-8")
        html_file.write_text(_rewrite_asset_paths(html, base_url), encoding="utf-8")
```

3. Add tests to `tests/test_builder.py`:
   - `test_rewrite_asset_paths_basic`: relative href/src gets prefixed
   - `test_rewrite_asset_paths_skips_absolute`: http/https/data/# URLs left unchanged
   - `test_rewrite_asset_paths_mixed`: HTML with both relative and absolute refs, only relative rewritten
   - `test_build_with_base_url`: Full build with `base_url="https://cdn.example.com/deck/abc"`, verify `slide_01.html` contains `href="https://cdn.example.com/deck/abc/theme/variables.css"` and `present.html` contains absolute slide refs
   - `test_build_without_base_url`: Build with no base_url, verify paths remain relative (regression guard)

Use existing test patterns: `tmp_path` fixture, minimal config with title+closing slides, `PresentationBuilder(config_path=..., metrics_path=...)`.
