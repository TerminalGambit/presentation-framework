  <name>Task 1: Create platform package with storage, worker, and FastAPI app</name>
  <files>platform/__init__.py, platform/storage.py, platform/worker.py, platform/api.py, setup.py</files>
  <action>
1. Create `platform/__init__.py` with a module docstring: `"""Hosted platform for sharing and serving presentation decks."""`

2. Create `platform/storage.py`:
   - `STORE_DIR = Path("data")` module-level constant (overridable in tests)
   - `def init_storage(store_dir: Path | None = None) -> Path`: creates the data dir, returns it
   - `def store_deck(config_bytes: bytes, metrics_bytes: bytes, store_dir: Path | None = None) -> tuple[str, Path]`: generates `deck_id = str(uuid.uuid4())`, creates `store_dir / deck_id /`, writes `presentation.yaml` and `metrics.json` as bytes, returns `(deck_id, deck_dir)`
   - `def get_deck_dir(deck_id: str, store_dir: Path | None = None) -> Path | None`: returns `store_dir / deck_id` if it exists, else None
   - `def delete_deck(deck_id: str, store_dir: Path | None = None) -> bool`: removes deck dir if exists, returns True/False
   - `def get_slides_dir(deck_id: str, store_dir: Path | None = None) -> Path | None`: returns `deck_dir / "slides"` if the build output exists

3. Create `platform/worker.py`:
   - `def run_build(deck_dir: Path, base_url: str | None = None) -> dict`: imports PresentationBuilder, constructs with `config_path=str(deck_dir / "presentation.yaml")`, `metrics_path=str(deck_dir / "metrics.json")`. Calls `builder.build(output_dir=str(deck_dir / "slides"), base_url=base_url)`. Returns `{"slide_count": N, "warnings": builder._warnings, "contrast_warnings": builder._contrast_warnings}`. Catches exceptions and returns `{"error": str(e)}`.

4. Create `platform/api.py`:
   - Import FastAPI, CORSMiddleware, Request, UploadFile
   - Import slowapi: `Limiter, _rate_limit_exceeded_handler, get_remote_address, RateLimitExceeded`
   - Create `limiter = Limiter(key_func=get_remote_address)`
   - Create `app = FastAPI(title="Presentation Platform API", version="1.0.0")`
   - Wire `app.state.limiter = limiter` and `app.add_exception_handler(RateLimitExceeded, _rate_limit_exceeded_handler)`
   - Add CORSMiddleware with `allow_origins=["*"], allow_methods=["*"], allow_headers=["*"]`
   - Add `EmbedHeadersMiddleware` (BaseHTTPMiddleware subclass) that sets `Content-Security-Policy: frame-ancestors *` on responses where `request.url.path.startswith("/d/")`
   - Pydantic response models:
     - `BuildResponse(deck_id: str, url: str, slide_count: int, warnings: list = [])`
     - `ValidateResponse(valid: bool, errors: list[str] = [])`
     - `EmbedResponse(deck_id: str, iframe_html: str, url: str)`
   - `POST /api/build` (rate limited "10/minute"):
     - Accepts `config: UploadFile`, `metrics: UploadFile`
     - Reads bytes immediately (UploadFile closed-before-BG-task pitfall)
     - Calls `store_deck(config_bytes, metrics_bytes)`
     - Calls `run_build(deck_dir, base_url=f"/d/{deck_id}")` synchronously via `asyncio.get_event_loop().run_in_executor(None, ...)`
     - If error in result, returns 422 with error detail
     - On success: mounts deck slides as StaticFiles at `/d/{deck_id}` with `html=True`
     - Returns `BuildResponse(deck_id=deck_id, url=f"/d/{deck_id}/present.html", slide_count=result["slide_count"])`
   - `POST /api/validate` (rate limited "30/minute"):
     - Accepts `config: UploadFile`
     - Reads bytes, writes to temp file, loads via PresentationBuilder, calls `validate_config()`
     - Returns `ValidateResponse(valid=len(errors)==0, errors=errors)`
   - `GET /api/decks/{deck_id}/embed`:
     - Returns `EmbedResponse` with iframe HTML snippet:
       ```html
       <iframe src="/d/{deck_id}/present.html" width="1280" height="720" allowfullscreen style="border:none;max-width:100%;aspect-ratio:16/9;"></iframe>
       ```
   - `DELETE /api/decks/{deck_id}`:
     - Calls `delete_deck(deck_id)`, returns 200 or 404

   IMPORTANT: Register all API routes FIRST, then mount StaticFiles directories. This avoids FastAPI's first-match routing issue where a `/d/` mount catches API routes.

5. Update `setup.py`:
   - Add `"platform": ["fastapi>=0.110", "uvicorn>=0.27", "python-multipart>=0.0.9", "slowapi>=0.1.9"]` to `extras_require`
   - Add `"dev": ["httpx>=0.27", "pytest-asyncio>=0.23"]` to extras_require for test deps

2. Add a `serve` command under the `platform` group:
```python
@platform.command()
@click.option("--host", default="0.0.0.0", help="Bind host")
@click.option("--port", "-p", default=8000, help="Port number")
@click.option("--reload", is_flag=True, default=False, help="Enable auto-reload for development")
def serve(host: str, port: int, reload: bool):
    """Start the platform API server."""
    try:
        import uvicorn
    except ImportError:
        click.echo("Platform server requires FastAPI + uvicorn. Install with:")
        click.echo("  pip install presentation-framework[platform]")
        raise SystemExit(1)

    click.echo(f"Starting platform server at http://{host}:{port}")
    click.echo(f"API docs: http://{host}:{port}/docs")
    click.echo("Press Ctrl+C to stop.\n")
    uvicorn.run("platform.api:app", host=host, port=port, reload=reload)
```

3. Note: The lazy import pattern (`try: import uvicorn except ImportError`) is consistent with the existing pdf/pptx CLI commands. The `platform.api:app` import string lets uvicorn handle the import so --reload works correctly.
  </action>

  <done>`pf platform serve` command exists, shows help with --host, --port, --reload options, and requires pf[platform] deps.</done>
