  <name>Task 1: Create analytics SQLite store and wire API endpoints</name>
  <files>platform/analytics.py, platform/api.py</files>
  <action>
1. Create `platform/analytics.py`:
   - `DB_PATH: Path = Path("data/analytics.db")` (module-level, overridable for tests)
   - `def init_db(db_path: Path | None = None)`: Creates `slide_events` table if not exists with columns: `id INTEGER PRIMARY KEY AUTOINCREMENT`, `deck_id TEXT NOT NULL`, `slide_idx INTEGER NOT NULL`, `duration_ms INTEGER NOT NULL`, `ts REAL DEFAULT (unixepoch('now', 'subsec'))`. Uses `sqlite3.connect(db_path or DB_PATH)`.
   - `def record_event(deck_id: str, slide_idx: int, duration_ms: int, db_path: Path | None = None)`: INSERT into slide_events. Calls `init_db(db_path)` to ensure table exists (idempotent).
   - `def get_dashboard(deck_id: str, db_path: Path | None = None) -> list[dict]`: SELECT aggregation — `slide_idx, COUNT(*) as views, SUM(duration_ms) as total_ms, AVG(duration_ms) as avg_ms` grouped by slide_idx, ordered by slide_idx. Returns list of dicts.
   - `def get_total_views(deck_id: str, db_path: Path | None = None) -> int`: Returns COUNT of distinct rows for the deck.

2. In `platform/api.py`, add:
   - Pydantic models:
     - `EventPayload(deck_id: str, slide: int, duration_ms: int)`
     - `DashboardEntry(slide: int, views: int, total_ms: int, avg_ms: float)`
     - `DashboardResponse(deck_id: str, total_views: int, slides: list[DashboardEntry])`
   - `POST /api/events` endpoint: Accepts JSON body `EventPayload`. Calls `record_event(...)`. Returns `{"status": "ok"}`. No rate limiting on this endpoint (beacons are high-frequency by design).
   - `GET /api/decks/{deck_id}/dashboard` endpoint: Calls `get_dashboard(deck_id)` and `get_total_views(deck_id)`. Returns `DashboardResponse`.
   - Call `init_db()` in a FastAPI `@app.on_event("startup")` handler (or use lifespan context manager if preferred).

3. Add a JS beacon script snippet as a constant string in `platform/api.py` (or `platform/analytics.py`):
   ```python
   BEACON_SCRIPT = """
   <script>
   (function() {
     var analyticsUrl = window.__PF_ANALYTICS_URL;
     var deckId = window.__PF_DECK_ID;
     if (!analyticsUrl || !deckId) return;
     var slideStart = Date.now();
     var currentSlide = 1;
     function sendEvent(slideIdx, durationMs) {
       navigator.sendBeacon(analyticsUrl,
         new Blob([JSON.stringify({deck_id: deckId, slide: slideIdx, duration_ms: durationMs})],
         {type: 'application/json'}));
     }
     document.addEventListener('visibilitychange', function() {
       if (document.visibilityState === 'hidden') sendEvent(currentSlide, Date.now() - slideStart);
     });
     var _origShow = window.show;
     if (_origShow) {
       window.show = function(n, instant, allFrags) {
         sendEvent(currentSlide, Date.now() - slideStart);
         currentSlide = n; slideStart = Date.now();
         _origShow(n, instant, allFrags);
       };
     }
   })();
   </script>
   """.strip()
   ```
   This script will be injected by the platform into served `present.html` files in a future enhancement. For now, document it as the client-side pattern and expose it via `GET /api/beacon-script?deck_id=X` endpoint that returns the script with `__PF_ANALYTICS_URL` and `__PF_DECK_ID` set.
